## Documentation



### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requrements.txt
```


### How To Start
```shell
$ application.java
```


### Dependency

- java==11
- SpringBoot==2.7.10



### Features

- 행운의 숫자 추출
- 행운의 숫자 저장
- 행운의 숫자 기반 구매
